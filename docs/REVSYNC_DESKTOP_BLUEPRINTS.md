# RevSync Pro Blueprints

## Start Center
- Hero region with product promise, readiness metrics, and three primary actions.
- Recent projects grid with validation score, package state, and last-opened metadata.
- Utility column for import, diagnostics, and recovery entry points.
- Marketplace queue preview for package-oriented workflows.

## Active Workspace
- Left project tree for sections and artifacts.
- Top tab strip for `overview`, `map`, `validation`, and `publish`.
- Central canvas for task-specific content.
- Right inspector for metadata, policy state, and safety messaging.
- Bottom output panel for logs and low-noise operational feedback.

## New Tune Project Wizard
- Step 1: deterministic vehicle selection
- Step 2: stock file import/reference capture
- Step 3: ECU profile identification
- Step 4: metadata and safeguards
- Output: project record that opens directly into the active workspace

## Validation Report
- Lead with score and readiness state.
- Blocking issues shown before warnings.
- Compatibility, signing, and recovery checks stay visible in summary.
- Release logic is severity-led instead of card-driven.

## Publish Flow
- Package summary on the left.
- Release notes and publish controls on the right.
- Signature and validation state visible before confirmation.
- Publish remains explicit and calm, never celebratory.
